## 4. R01 Eval 上传包要求

R01 如果要 official eval，必须上传新 eval 目录，至少包含：

```text
eval/dataset.py
eval/model.py
eval/infer.py
```

原因：R01 新增了：

```text
seq_recent_stats
use_seq_recent_stats
seq_recent_stats_dim
seq_recent_stats_gate_init
seq_recent_stats projector/gate 参数
```

旧 eval 可能不产出 `seq_recent_stats`、不认识 projector/gate、不读取 train_config，导致 strict load missing/unexpected key 或构建错误模型。

D02 只改 `seq_encoder_type=swiglu`，理论上旧 eval 已支持；但使用当前新 eval 也可以。关键是 D02 checkpoint 的 `train_config.json` 必须为：

```text
seq_encoder_type=swiglu
user_dense_projector_type=grouped
use_time_context=0
use_seq_recent_stats=0
```

结论：

```text
D02 eval：新 eval 可以用，旧 eval 理论也能用。
R01 eval：必须用新 eval。
```
