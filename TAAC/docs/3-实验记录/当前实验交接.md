## 4. train/eval 同步边界

只改这些通常不需要更新 eval：

- lr / sparse_lr / warmup。
- loss。
- valid split。
- checkpoint 选择。
- patience / num_epochs。

这些必须同步 eval：

- 新增 `ModelInput` 字段。
- 新增 dataset 输出张量。
- 新增模型结构参数。
- 修改序列特征构造、time bucket、OOV/clip 逻辑。
