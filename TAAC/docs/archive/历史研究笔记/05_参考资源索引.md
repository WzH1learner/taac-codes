## 下一步学习路径建议

### 阶段 1：跑通 Baseline（当前）
1. 上传 train/ 到 AngelML 平台
2. 跑通训练 → 发布模型 → 跑通评估
3. 确认 predictions.json 格式正确

### 阶段 2：理解模型
1. 精读 `model.py`：PCVRHyFormer 架构
2. 精读 `dataset.py`：数据加载与 feature schema
3. 精读 `trainer.py`：训练循环与优化策略

### 阶段 3：调参优化
1. 调整 `d_model`, `num_hyformer_blocks`, `num_queries`
2. 尝试 `seq_encoder_type` 三种变体
3. 调整 `seq_max_lens` 截断长度
4. BCE vs Focal Loss 对比

### 阶段 4：模型改进
1. 特征工程：新特征组合、交叉特征
2. 序列建模：更长序列、分层注意力
3. 损失函数：尝试 Listwise/Pairwise
4. 集成学习：多模型融合

### 阶段 5：工程优化
1. 分布式训练加速
2. 模型压缩与量化
3. 推理性能优化
