# TAAC2026 竞赛概述与技术趋势

## 竞赛基本信息

- **全称**：2026 腾讯统一推荐挑战赛（2026 TENCENT UNI-REC CHALLENGE）
- **主办方**：腾讯广告 × KDD（ACM SIGKDD Conference）联合举办
- **总奖金**：600 万元人民币
- **学术赛道冠军**：30 万美元
- **工业赛道冠军**：15 万美元
- **专项创新奖**：9 万美元（Scaling Law + 统一架构方向）

## 核心赛题

**"面向大规模推荐的序列建模与特征交互的统一"**

- 在一个模型架构中同时处理：**多字段序列特征** + **非序列特征**
- 输出目标广告的 **pCVR**（post-click conversion rate，点击后转化率）
- 突破传统"检索-排序-漏斗"的多阶段级联策略

## 技术趋势：从判别式到生成式

> 推荐系统正从"判别式"走向"生成式 Agent"的范式更迭。

### 过去（判别式推荐）
- 逻辑：检索 → 排序 → 漏斗
- 特征工程主导，比谁融合更多有效信息
- 基于历史数据的机械匹配：你点了 A，我推 B

### 现在（生成式推荐）
- 全模态序列生成式推荐（AMGR）
- 基于用户的全模态历史行为（文本、视觉、协同行为等）
- 精准预测下一次交互的广告内容
- 考虑新鲜度特征、交叉特征组合等复杂因素
- 实现从"识别"到"创造"的范式转变

## 技术难点

1. **多模态数据融合**：文本、图像、协同行为等多模态信息统一建模
2. **长序列处理**：用户两年里的行为轨迹，超长序列建模
3. **特征异构统一**：序列特征和非序列特征在同一个架构中处理
4. **冷启动问题**：新用户/新广告缺乏历史数据
5. **工程可落地性**：工业级分布式训练、实时推理

## 参赛数据规模

- 覆盖近 30 个国家
- 340 余所中国高校 + 140 多所国际院校
- 8400 余名学生，2800 多支队伍

- GPU 全程护航
- 预装：Ubuntu 22.04, CUDA 12.6, cuDNN 9.5.1
