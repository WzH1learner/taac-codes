# 项目文档入口

本文档目录是 TAAC2026 PCVRHyFormer 项目的知识入口。所有协作者应优先阅读这里，再进入具体资料。

## 项目目标

我们要优化一个统一建模的 pCVR 推荐模型：同时利用用户/物品非序列特征与多域行为序列，在官方 eval AUC 上超过当前最佳结果。由于 daily eval 次数有限，文档体系的核心作用是避免重复实验、保留有效结论，并让每次代码改动都有清晰依据。

## 阅读顺序

1. [推荐算法优化蓝图](1-核心蓝图/推荐算法优化蓝图.md)：理解当前优化主线、优先级和风险。
2. [当前实验交接](3-实验记录/当前实验交接.md)：查看当前最佳配置、待办和不要再消耗 eval 的方向。
3. [平台规则与提交规范](2-参考资料/比赛平台规则与提交规范.md)：提交训练与 eval 前确认平台要求。
4. [样例数据说明](2-参考资料/样例数据说明.md)：需要改 dataset 或特征工程时阅读。

## 维护原则

- 活跃文档只保留当前有用、唯一、权威的内容。
- 已被蓝图吸收的阶段性笔记进入 `archive/`，不再作为决策入口。
- 新实验结果优先写入 `docs/3-实验记录/`，确认稳定后再回填蓝图。
- 新增模型输入或结构参数时，必须同时检查 `train/` 与 `eval/` 的兼容性。
